🚀 New Member? Follow These Steps First
  1. Open the share invite email and accept it, or go to onedrive.com → Shared
  2. Find the team folder → right-click → "Add shortcut to My files"
  3. Wait 1–2 min for it to appear in File Explorer → OneDrive
  4. In File Explorer, right-click the folder → "Always keep on this device" (wait for the green ✓)
  5. Come back here and click "Choose Shared Folder" below
  6. Navigate to that OneDrive folder and click Select Folder
  7. Click "Test Folder Access" to confirm it works
⚠️ The extension only works with a locally synced copy of the folder — it cannot access OneDrive on the web directly. The folder must be shared with you as "Can edit" (not "Can view").
📁 Shared OneDrive Folder

Choose the OneDrive folder that is shared with the team and synced to this PC. Each member writes to their own JSON file, and the admin dashboard merges those files for visibility.

After a browser restart, access must be re-granted. If you see “permission denied” or “not granted” after restarting Chrome or Edge, click “Re-grant Folder Access” below. It opens a full tab and asks you to allow access again — choose Edit. You will not need to re-pick the folder.

  • Each team member picks the same shared OneDrive folder
  • When you click "Submit", your entry is written into your own team file
  • Data is always saved locally first, then copied into the shared folder
  • No Azure, Graph API, or published website is required
