⏱ QC Time Tracker · Team Sync

Setup & Fix Guide

Everyone writes to their own file in one shared OneDrive folder, and the admin view merges them so the whole team sees every update. This guide makes that work for all 30 people — and fixes the “file is not granted to you” error.

No API needed No Power Automate Just OneDrive + Chrome / Edge
If it “isn’t working for everyone,” it is almost always one of three OneDrive gaps — not the extension. The folder was shared view-only, the member never added the shortcut, or the folder is still online-only. The three rules below close all three gaps. The new 🧪 Test Folder Access button (v2.2) will also tell each person exactly which one they hit.

The 3 rules that make or break it

1
Share with “Can edit” — never “Can view.”
View-only folders sync to the PC as read-only, so the extension can’t create the member’s file. This is the #1 cause of “not granted.”
2
Each member must “Add shortcut to My files.”
Until they do, the folder lives only on OneDrive-web and never lands on the PC, so the file picker can’t see it.
3
Each member must set “Always keep on this device.”
Online-only files (cloud icon) aren’t truly on disk. Reads/writes fail until the folder is pinned locally with a green check.
PART A

Create & share the folder

Admin · once
1

Make one shared folder in OneDrive

Do this from the OneDrive desktop folder on your own PC (File Explorer → OneDrive).

  1. Create a folder named exactly QC-Team-Tracker
  2. Right-click it → “Always keep on this device” (so your own copy is real and writable)
2

Share it with “Can edit”

  1. Right-click the folder → Share (or OneDrive → Share)
  2. Click the permission selector (it usually reads “Can view”) and change it to “Can edit”
  3. Add all 30 members by their work email, or create a “Can edit” link and send it
  4. Send
Check this twice. If a member ever sees “not granted” even after pinning the folder, their invite was almost certainly “Can view.” Re-share to that person with Can edit, then have them remove and re-add the shortcut.
3

Send members two things

  • The “Can edit” share link to QC-Team-Tracker
  • The extension ZIP + this guide
✓ That’s the entire admin job. You never have to touch Power Automate, the Graph API, or any automation. Members do the rest once each.
PART B

Connect on each PC

Every member · ~5 min
1

Add the folder to your OneDrive

  1. Open the share email and click Open — or go to onedrive.com → Shared
  2. Find QC-Team-Tracker → click “Add shortcut to My files”
  3. Wait 1–2 minutes
If you don’t see “Add shortcut to My files,” open the “⋯” menu on the folder, or check whether it’s already under File Explorer → OneDrive.
2

Pin it to your PC (the step people skip)

  1. Open File Explorer → OneDrive — you should now see QC-Team-Tracker
  2. Right-click it → “Always keep on this device”
  3. Wait until the folder shows a solid green check (not a cloud)
Skipping this causes the write to fail. A cloud icon means the files are online-only and not really on disk yet.
3

Load the extension (Chrome or Edge)

  1. Unzip the extension to a permanent folder (don’t delete it afterwards)
  2. Go to chrome://extensions  (on Edge use edge://extensions)
  3. Turn on Developer mode (top-right on Chrome, bottom-left on Edge)
  4. Click Load unpacked → select the unzipped folder
  5. Pin the ⏱ QC Time Tracker icon to the toolbar
✓ Microsoft Edge is fully supported — it’s built on the same engine as Chrome. Any “Chrome” wording in prompts applies to Edge too.
Desktop Chrome or Edge only. The folder-picker isn’t available in other browsers or on mobile.
4

Connect the folder & test

  1. Open the extension → sign in with your Name + Email
  2. Click ⚙️ Settings → 🔗 Open in Full Tab (the picker needs a full tab, not the small popup)
  3. Click 📁 Choose Shared Folder → in File Explorer go to OneDrive → QC-Team-Tracker → Select Folder → Allow
  4. Click 🧪 Test Folder Access
✓ Success looks like: “All good. The folder is writable and synced. Your team file: qc-time-tracker-team-yourname.json.” If you see anything else, the test now tells you the exact cause — see the table below.
DAILY

After every Chrome restart

5 seconds

For security, Chrome/Edge forgets folder access when it fully restarts. You do not re-pick the folder — just re-grant:

  1. Open the extension → ⚙️ Settings
  2. Click 🔑 Re-grant Folder Access — it opens a full tab (the prompt closes the small popup, so this is automatic)
  3. In that tab, click 🔑 Re-grant Folder Access again → when the prompt appears choose Edit / Allow (not “View only”)
Once per browser session, not every time you use it. Choosing Edit is what makes it stick — “View only” will keep failing.
FIX

Message → cause → fix

What the member seesWhat it means & how to fix it
“The folder is connected but the extension can’t write into it.” Read-only or online-only folder. 1) Confirm the admin shared it as Can edit (not Can view). 2) Right-click the folder in OneDrive → Always keep on this device, wait for the green check. 3) Make sure you picked the folder inside File Explorer → OneDrive, not a web page. Then test again.
“Permission was not granted” — Re-grant keeps failing no matter how many times I click The prompt is closing the small popup. The new build opens a full tab automatically when you click 🔑 Re-grant Folder Access. In that tab, click the button again and choose Edit on the prompt. (If you’re on an older build, click 🔗 Open in Full Tab first.) Also make sure you don’t pick “View only” on the prompt.
“Chrome/Edge needs you to allow access again” / “permission denied” after restart Normal after a browser restart. Click 🔑 Re-grant Folder Access, then choose Edit in the full tab. No need to re-pick the folder.
“No folder connected on this PC yet.” You haven’t selected the folder on this machine. Click 📁 Choose Shared Folder and pick OneDrive → QC-Team-Tracker.
The folder picker won’t open / the popup keeps closing The small popup can’t host the picker. Click 🔗 Open in Full Tab first, then 📁 Choose Shared Folder.
Folder isn’t in the picker at all It never synced down. Re-do Add shortcut to My files, wait 1–2 min, confirm it appears under File Explorer → OneDrive, then pin it.
“This browser can’t do folder sync.” You’re not in Chrome/Edge desktop. Open it in Google Chrome or Microsoft Edge.
I can write my file, but I don’t see other people’s entries Open the admin view and click ↻ Refresh Team Data. If still missing, the other members haven’t pinned/granted yet, or OneDrive is mid-sync — wait a minute and refresh.
Two people seem to overwrite each other They’re signed in with the same email. Each person must use their own email so they get their own file.
One-line health check for the admin: open File Explorer → OneDrive → QC-Team-Tracker. You should see one qc-time-tracker-team-*.json per active member, each with a green check. Missing names = those people haven’t finished Part B.
